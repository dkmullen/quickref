Learning the shell

ls

  • ls (lists filed in current dir)
  • ls -l (lists files in long format)
  • ls -a (lists all files including hidden)
  • ls -la (does both of the above)
  • ls /dir_name (lists files in another dir)
  • ls /dir1 /dir2 (lists files in multiple dirs)

cd

  • cd (by itself, changes to user's home dir)
  • cd - (changes to previous working dir)
  • cd .. (changes to the parent dir)
  • cd ./dir_name (changes to child dir - cd dir_name is almost always okay too)

Misc

  • up-arrow (show previous command)
  • pwd (print working dir)
  • less (show contents of a txt file
  • file file_name (identifies file type)

More on less

  • page up or b (scroll up one page)
  • page down or space (scroll down one page)
  • G (jump to end)
  • 1G (jump to beginning)
  • /characters (search forward for specified characters)
  • n (repeat search)
  • h (list commands and options for less)
  • q (quit)

Manipulating Files

  • cp (copy - but use cp -r dir1 dir2 to move dirs)
  • mv (move - use for renaming files too by moving it to the same place with a new name)
  • rm (remove)
  • mkdir (make a new dir)

Some wildcards

  • * (matches any character)
  • ? (matches any single character)
  • [characters] (matches any character that is part of the set, including a few special classes):
    • [:alnum:] alphanumeric
    • [:alpha:] alphabetic
    • [:digit:] numerals
    • [:upper:] uppercase letters
    • [:lower:] lowercase letters
  • [!characters] (matches any character that is NOT part of the set)

Usage examples

  • g* (all filenames starting with 'g')
  • b*.txt (all filenames starting with 'b' and ending with 'txt')
  • DATA??? (all filenames starting with 'DATA' followed by exactly three more characters)
  • [abc]* (all filenames starting with 'a' 'b' or 'c' followed by any other characters)
  • BACKUP.[[:digit:]][[:digit:]] (all filenames starting with BACKUP followed by exactly two numerals)
  • *[![:lower:]] (all filenames that do NOT end with a lowercase letter)
