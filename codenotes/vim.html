VIM Notes

Command Mode: Moving

  • h - Moves one char to the left
  • l - Moves one char to the right
  • j - Moves down one line
  • k - Moves up one line
  • 0 - Moves to beginning of line (zero!)
  • $ - Moves to end of line
  • w - Moves forward one word
  • b - Moves backward one word
  • G - Moves to end of file
  • gg - Moves to beginning of file
  • `. - Moves to last edit

AND prefacing a command with a number executes that command multiple times.

Command Mode: Editing

  • d - Starts delete operation
  • dd - Cuts a line, puts it into buffer
  • dw - Deletes a word
  • d0 - Deletes to start of line(zero!)
  • d$ - Deletes to end of line
  • dgg - Deletes to beginning of file
  • dG - Deletes to end of file
  • u - Undo
  • CTRL-r - Redo last undo
  • V - Highlight one line at a time (then use arrow to highlight more)
  • v - Highlight one char at a time (then use arrow to highlight more)
  • y - Yank text into copy buffer
  • P - Paste text on the current line
  • p - Paste text after the current line

Command Mode: Searching and replacing

  • /bunny - Search forward in doc for 'bunny'. Use n to jump to the next one, N for the previous one.
  • ?bunny - Search backward in doc for 'bunny'
  • :%s/bunny/rabbit/g - Search entire doc for 'bunny', replace w/ 'rabbit'
  • :%s/bunny/rabbit/gc - Same as above but confirm before replacing
