Powershell: One-line list

  • pwd - Print working directory
  • hostname - My computer's network name
  • mkdir - Make directory
  • cd - Change directory
  • ls - List directory
  • rm - Remove
  • rmdir - Remove directory (Is this still needed?)
  • pushd - Push directory
  • popd - Pop directory
  • cp - Copy file or directory
  • robocopy - Robust copy
  • mv - Move file or directory
  • more - Page through a file
  • type - Print the whole file
  • forfiles - Run commands on lots of files
  • dir -r - Find files
  • select-string - Find things inside of files
  • help - Read a manual page
  • helpctr - Find what man page is appropriate
  • echo - Print some arguments
  • set - Export/set a new environment variable
  • exit - Exit the shell
  • runas - Become root user
  • attrib - Change permission modifiers
  • iCACLS - Change ownership
  • up-arrow - Show previous command
