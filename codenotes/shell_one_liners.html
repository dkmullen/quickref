Learning the shell: One-line list

  • alias - Assign commands & values to new or existing commands (alias: foo='ls -l ~/MyDocs')
  • apropos - Find what man page is appropriate
  • cal - Show a calendar of the current month
  • cat - Concatenate files and print to standard output
  • -A - Show non-printing characters, such as TAB
  • -n - Show line numbers
  • -s - Supress multiple blank lines
  • cd - Change directory
  • chmod - Change permission modifiers
  • chown - Change ownership
  • clear - Clear the terminal (also CTRL-L)
  • cp - Copy file or directory
  • date - Show the date
  • df - Show free space on discs
  • echo - Print some arguments
  • env - Look at your environment
  • exit or CTRL-D - Exit the shell
  • export - Export/set a new environment variable
  • find - Find files across the whole system with lots of modifiers
  • file - Identifies file type
  • free - Show free memory
  • grep - Find things inside of files
    • -i or --ignore-case - Ignore case (default is to NOT ignore case)
    • -v or --invert-match - Print only lines that do not match the pattern
    • -c or --count - Print the number of matches (or non-matches, if v is specified) rather than the matches themselves.
    • -l or --files-with-matches - Print the name of the file containing a match, not the match itself.
    • -L or --files-without-match - Print the name of the files that do NOT contain a match.
    • -n or --line-number - Print each matching line with the line number in the file.
    • -h or --no-filename - For multi-file searches, supress filenames.
  • head - Print first ten lines of a file
  • hostname - My computer's network name
  • less - Page through a file
  • ln - Make hard and symbolic links
  • locate - Search entire system for files and dirs by name.
  • ls - List directory
  • man - Read a maunal page
  • mkdir - Make one directory (use -p for a path)
  • mv - Move file or directory
  • popd - Pop directory
  • pushd - Push directory
  • pwd - Print working directory
  • rm - Remove
  • rmdir - Remove directory (Is this still needed?)
  • sort - Sort lines of text
  • -b - Ignore leading blanks
  • -f - Ignore case
  • -n - Numeric sort
  • -r - Reverse sort
  • sudo - Become root user
  • tail - Print last ten lines of a file
  • touch - Make a new file or update timestamp of existing one.
  • up-arrow - Show previous command
  • xargs - Execute arguments
  • > - Redirects the output of the command, creating or overwriting
  • >> - Redirects the output of the command, appending
  • | - Redirects the output of the command, piping it to another command

Notes:

  • pushd - Takes your current directory and "pushes" it into a list for later, then it changes to another directory. It's like saying, "Save where I am, then go here."
  • popd - Takes the last directory you pushed and "pops" it off, taking you back there.
  • If you run pushd with no arguments, it will switch between your current directory and the last one you pushed. It's an easy way to switch back and forth between two directories. This does not work in PowerShell.

Keyboard shortcuts in Shell

  • CTRL-A - Move cursor to beginning of line
  • CTRL-C - Quit process
  • CTRL-D - Quit the terminal
  • CTRL-E - Move cursor to end of line
  • ALT-F - Move cursor forward one word
  • ALT-B - Move cursor backward one word
  • clear - Clear the terminal (also CTRL-L)
  • CTRL-K - Kill (cut) text from cursor to end of line
  • CTRL-U - Kill text from cursor to beginning of line
  • ALT-D - Kill text from cursor to end of current word
  • ALT-BACKSPACE - Kill text from cursor to beginning of current one word
  • CTRL-Y - Yank text from the kill-ring and paste it at the cursor
